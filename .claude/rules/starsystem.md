# STARSYSTEM

This repo is a STARSYSTEM. Before working:

1. Equip persona `starship-pilot-heaven-framework-repo`
2. `orient()` to check project status
3. `start_starlog()` to begin a session
4. `fly()` to browse available flight configs
5. Log insights/bugs/decisions to debug diary as you work
6. `end_starlog()` when session is done
